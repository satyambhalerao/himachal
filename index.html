One-time setup — takes about 5 minutes

📋 Setup Steps

1
Go to console.firebase.google.com → Create project (disable analytics if asked)
2
In your project → Firestore Database → Create database → choose test mode
3
Storage → Get started → test mode (for photo uploads) — optional, skip if it asks you to upgrade plan
4
Project Settings ⚙️ → Your apps → Add web app → copy the firebaseConfig object
5
Open index.html in any text editor → find FIREBASE_CONFIG near the top of the <script> → paste your values
6
Also set ADMIN_PASSWORD in the same section
7
Deploy to Vercel — drag & drop the file (or push to GitHub)
Tip: Once deployed, share the Vercel URL with everyone in the group — all data syncs live across all devices!
